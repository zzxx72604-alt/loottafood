LoottaFood — where to drop your images
======================================
Put image files in THIS folder:  frontend/src/assets/

1) ScanMe order QR  (shows when you tap "ScanMe" on phone/tablet)
   File name:  scan-qr.jpg
   -> Save your QR image here exactly as: scan-qr.jpg

2) Logo  (future: replace the "LoottaFood" text logo in the header)
   Suggested file name:  logo.png   (transparent PNG, ~200x60)
   -> Tell me when added and I'll wire it into the header.

3) Default profile icon  (future: profile avatar)
   Suggested file name:  profile-default.png   (square, e.g. 256x256)
   -> Tell me when added and I'll wire it into the profile page.

Note: after adding/replacing a file here, hard-refresh the browser
(Ctrl+Shift+R) so it isn't showing a cached version.
